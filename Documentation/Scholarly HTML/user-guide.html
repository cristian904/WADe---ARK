Case study 1 (Exploring museums)

This case study shows how users can explore museums from all over Romania. The ARK interface presents a visualisation of the country with all museums spread across the map. A user can zoom in on the city they like and select which museum galery he wants to visit. He will click on the museum location and them view all the artwork from the gallery (Figure 1).

The user will be redirected to a new page where he can see different statistics on the selected museum and also all the works of art available. Statistics:

  • Pie chart describing the state of the artworks: (good state, bad state, mediocre state etc.) (Figure 2)
  • Bar plot showing the artists present in the museum and how many works they have there. (Figure 3)
  • Bar plot describing what types of work is available in the museum and how many of them. (Figure 4)

Bellow these statistics you can see all the available artwork from the selected museum. Hovering over the image will give you some general information and then the user can click on it and get full details about the artwork. (Figure 5)

On the art details page the user can find out general knowledge about the selected piece of art like author, year, museum in which you can see it, type of work, real size and a short description (Figure 6). Below you can see two lists of recommended art works (available through a carousel type scroll) witch shows art from the same author (Figure 6) and art from the same museum (Figure 7).

Case study 2 (Exploring artists)

This scenario presents how a user can browse artists and their artworks. By selecting the "Artiști" tab from the page header you can see a table with all artists present in the romanian museums (Figure 8). You can also filter them by name if you want to find a specific author. (Figure 9)

Now the user can click on the details icon and navigate to the page of the artist where he can see some general information about him (birth date, death date, currents) and statistics about the museums where he can be found. (Figure 10) Bellow there is a list of all the artwork from this author (Figure 10) and some recommendations of authors from the same currents.(Figure 11)

Case study 3 (Exploring artwork)

So far we have seen how the user can browse museums and artists, but he can also look-up art. By clicking the "Artă" tab in the header of the page, the user can see a table with all available romanian artworks. (Figure 12) He also has the posibily of filtering them by name, author and type of work. (Figure 13)

Now the user can click the detail icon from any piece of art and navigate to the artwork page which was descibed in scenario 1. (Figure 6)
